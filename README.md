# Kurs Logische Labyrinthe SoSo 2026 HSD PBSA

Das Repository enthält das Beispielprogramm zum Kurstext, sowie das in dem Kurs entstandenen Interactive Fiction Spiel "Treibsand", das von den Studierenden im Verlauf des Sommersemester 2026 an der HSD PBSA FB 6 entworfen und programmiert wurde.

## Installation und Ausführung

### Scryer Prolog
Zur Ausführung wir scryer prolog benötigt. 
Dazu den Installationsanweisungen [hier](https://www.scryer.pl/) folgen.

Nach dem Download dieses Repositories in Terminal den entsprechenden Unterodrdner wechseln.

### Das Insitut (Beispielprogramm zum Skript)

Um das Beispielprogramm zu laden und auszuführen:

`scryer-prolog das_institut.pl`

`play.`Startet das Spiel.

Alternativ im Prolog REPL:

`consult(das_institut)`


### Treibsand (Semesterarbeit)

Um das Semesterspiel zu laden und auszuführen:

`scryer-prolog treibsand.pl`

`play.`Startet das Spiel.

Alternativ im Prolog REPL:

`consult(treibsand)`
